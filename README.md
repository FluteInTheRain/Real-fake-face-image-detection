# ASSIGNMENT 2: INVESTIGATING STABLE DIFFUSION MODELS AND FAKE DATA DETECTION/PREVENTION SOLUTIONS
## PART 1: STABLE DIFFUSION MODEL INVESTIGATION
- This repository investigates the SSD-1B-Traditional (short for Segmind Stable Diffusion 1B) model, which is a "distilled" version of the massive Stable Diffusion XL (SDXL) 1.0. While SDXL has approximately 2.6 billion parameters, SSD-1B contains only about 1.3 billion. Its objective is to retain 90-95% of SDXL's image quality while doubling processing speed and significantly reducing required VRAM, making it ideal for research purposes.
- The investigation is conducted through two experiments:
### Experiment 1: 
**Research question**: At what number of inference steps does the SSD-1B model reach convergence (SSIM >= 0.95 relative to the 50-step reference), while minimising latency?
### Experiment 2:
**Research question**: How does the Classifier-Free Guidance (CFG) scale affect prompt adherence (CLIP Score) and over-saturation risk (HSV saturation channel) in SSD-1B?
## PART 2: PROPOSED APPLICATION FOR REAL/FAKE IMAGE DETECTION
**To simplify the problem, we focus on identifying whether a human face is a real photo or generated by AI models.**
- **Dataset**: Utilizing the "130k Real vs Fake Face" dataset from Kaggle: https://www.kaggle.com/datasets/shreyanshpatel1/130k-real-vs-fake-face
- **Architecture**: We utilize **ResNet-50**, a renowned Convolutional Neural Network (CNN) architecture primarily used for image recognition and feature extraction.
- **Methodology**: We employ Transfer Learning by using a pre-trained model and modifying the final fully connected layer from 1,000 outputs to 2 outputs (Real/Fake).
- **Optimization**: We "freeze" the initial parameters, allowing the model to only learn the weights connecting the final two layers. This approach reuses pre-trained features, significantly saving training time while leveraging the existing knowledge base of ResNet-50.
- **Outcome**: A robust model capable of distinguishing between real and synthetic human faces.
