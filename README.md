## PART 1: STABLE DIFFUSION MODEL INVESTIGATION
This repository investigates the SSD-1B-Traditional (Segmind Stable Diffusion 1B) model, a distilled version of Stable Diffusion XL (SDXL) 1.0. While SDXL has ~2.6 billion parameters, SSD-1B has ~1.3 billion, aiming to retain 90-95% image quality while doubling processing speed and significantly reducing VRAM requirements for research purposes.

The investigation is conducted through two experiments:

### Experiment 1: Convergence and Inference Steps
**Research question**: At what number of inference steps does the SSD-1B model reach convergence (SSIM >= 0.95 relative to the 50-step reference), while minimizing latency?

**Result**:
- **Drafting Configuration:** 12 steps is optimal (under 10s latency, early structure formation).
- **Production Configuration:** 20 steps is the best trade-off (~14-15s on a T4 GPU). SSIM peaks or stabilizes here. Higher steps (30-50) offer marginal visual benefits and waste VRAM.
> ![Experiment 1 Results](/assets//images/exp1result.png)

### Experiment 2: Classifier-Free Guidance (CFG) Impact
**Research question**: How does the CFG scale affect prompt adherence (CLIP Score) and over-saturation risk (HSV saturation channel) in SSD-1B?

**Result**: 
- **Safe Zone:** CFG 7.0 provides the highest average CLIP score before structural artifacts appear, keeping saturation (~90) within a highly safe range. It serves as an ideal default.
- **Style-based Adjustments:**
    - *Photorealistic Portraits*: Lower CFG (3.0 - 5.0) to preserve biological face structures and avoid artificial looks.
    - *Vector / 2D Flat Design / Logo*: Higher CFG (9.0 - 12.0) is safe, maximizing prompt adherence without color corruption.
    - *High Contrast Scenery / Cyberpunk*: Keep at 7.0 to avoid over-saturation.
> ![Experiment 2 Results](/assets/images/exp2result.png)

## PART 2: PROPOSED APPLICATION FOR REAL/FAKE IMAGE DETECTION
**To simplify the problem, we focus on identifying whether a human face is a real photo or generated by AI models.**

- **Dataset**: Utilizing the "130k Real vs Fake Face" dataset from Kaggle: [Link to Dataset](https://www.kaggle.com/datasets/shreyanshpatel1/130k-real-vs-fake-face)
- **Architecture**: **ResNet-50**, a renowned Convolutional Neural Network (CNN) architecture primarily used for image recognition and feature extraction.
- **Methodology & Optimization**: We employ Transfer Learning by modifying the final fully connected layer from 1,000 to 2 outputs (Real/Fake). By "freezing" the initial parameters, the model only learns the final layer weights. This reuses pre-trained features, dramatically saving training time while leveraging ResNet-50's knowledge.
- **Outcome**: A robust model capable of distinguishing between real and synthetic human faces.

## HOW TO INSTALL

### Prerequisites
- Install all necessary libraries via `requirements.txt`:
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
- Download the dataset from Kaggle and save it to `datasets/dataset.zip`: [https://www.kaggle.com/datasets/shreyanshpatel1/130k-real-vs-fake-face](https://www.kaggle.com/datasets/shreyanshpatel1/130k-real-vs-fake-face)

- Download the model SSD-1B here: [https://huggingface.co/segmind/SSD-1B](https://huggingface.co/segmind/SSD-1B). Save as folder `./SSD-1B-Traditional`.


### Running Part 1 (Stable Diffusion Investigation):
#### Experiment 1
- Run command: `python sd_research/experiment1_steps.py`
- Outputs:
    - `outputs/exp1_individual_prompts.png`
    - `outputs/exp1_average.png`
    - `outputs/exp1_generated_images/` (all generated PNGs)

#### Experiment 2
- Run command: `python sd_research/experiment2_cfg.py`
- Outputs:
    - `outputs/exp2_individual_prompts.png`
    - `outputs/exp2_average.png`
    - `outputs/exp2_generated_images/`

#### Displaying results in a grid:
- Run command: `python sd_research/make_grid.py`
- For a specific experiment:
    - `python sd_research/make_grid.py --exp 1`
    - `python sd_research/make_grid.py --exp 2`

#### Running both experiments simultaneously:
- Run command: `python sd_research/run_all.py`

### Running Part 2 (Real/Fake Face Detection App):
- Execute the `fake-image-detection/main.ipynb` notebook to train and generate the model.
- The trained model is saved as: `fake-image-detection/resnet50_best_real_fake.pth`
- To launch the web application (Streamlit): 
```bash
streamlit run fake-image-detection/app.py
```
